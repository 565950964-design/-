新手教程

从 0 到 1 使用这份账本：网页、微信、审批、撤销、导出

1. 快速开始（网页）

  1. 打开首页，默认账本是 web-local。
  2. 在“智能记账”输入一句话，比如“午饭 22 元”。
  3. 去“账单明细”核对记录，支持搜索、筛选、编辑、删除。
  4. 去“预算管理”设置本月预算，观察预算进度和日均支出。

2. 微信接入与审批

  1. 微信测试号 URL 填：你的域名/wechat，Token 填 WECHAT_TOKEN。
  2. 新用户先发“申请使用”。
  3. 管理员到“用户管理”输入 ADMIN_WEB_TOKEN 后审批。
  4. 审批通过后用户即可在微信自然语言记账。

3. 常用对话指令

  • 记账：吃饭 18 元 / 工资 8000
  • 查询：今天、昨天、本月、上月、查餐饮
  • 区间：上周、3月、2026年3月、3月1日到3月15日消费
  • 会返回该时间段总支出/总收入/结余和支出构成占比
  • 关键词：查滴滴 / 统计星巴克（支持某月范围）
  • 趋势：本周趋势（按周一到今天逐日输出）
  • 纠错：撤销上一笔；如误删，可发送恢复上一笔
  • 可视化：发送“可视化”/“看图表”，会返回网页看板链接

4. 数据管理

  • 账单明细可导出 CSV（Excel 直接打开）。
  • 不同 user_id 的账本相互隔离。
  • 审批日志用于追踪批准/拒绝操作。

5. 常见问题

  • 看不到用户管理数据：先输入管理员口令登录。
  • 微信校验失败：检查 URL 可公网访问、Token 一致。
  • 误记了一笔：在聊天里点“撤销这笔”或输入“撤销上一笔”。
